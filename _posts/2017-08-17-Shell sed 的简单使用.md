# Shell sed 的简单使用


```
sed -i "" 's/aaa/bbbb/g' /Users/Yunis/Desktop/XXX.py
```

这句shell 命令的意思是，查找到 aaa 把它替换为 bbbb 
`/Users/Yunis/Desktop/XXX.py` 是要替换文本的路径。

`""` 表示要备份的路径，为空表示不备份。
`s` :替换
`g` :全部替换标记
