#### 安装方式
1. $ git clone https://github.com/yunisSong/YunisTool.git
2. swift package generate-xcodeproj
2. $  swift build
3. $ swift build -c release
4. $ cp ./.build/release/BlogTool /usr/local/bin/BlogTool


#### 使用方式
$ BlogTool -t 使用Swift做一些重复性的工作 -tags Swift
