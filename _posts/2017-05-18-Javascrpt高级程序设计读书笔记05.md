`arguments.callee` 指向正在执行的函数的指针。

```
function factorial(num){    if (num <= 1){        return 1;    } else {        return num * arguments.callee(num-1);    } 
}
```

```
var factorial = (function f(num){        if (num <= 1){            return 1;        } else {            return num * f(num-1);    } 
});
```
